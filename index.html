Exploring a Simple Function: \( f(x) = x^2 + 3x + 2 \)

A mathematical function is a relationship between inputs and outputs. Let's explore the function: \[ f(x) = x^2 + 3x + 2 \] This is a quadratic function, and its graph is a parabola. Below, you can see the Python code to implement this function.

Python Implementation


def f(x):
    return x**2 + 3*x + 2

# Example usage:
x_values = range(-10, 11)
for x in x_values:
    print(f"f({x}) = {f(x)}")

This code defines the function \( f(x) \) and calculates its value for \( x \) in the range \(-10\) to \(10\).
