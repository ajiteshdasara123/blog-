Understanding the Pythagorean Theorem

One of the most famous theorems in mathematics is the Pythagorean Theorem. It states that in a right triangle, the square of the length of the hypotenuse (\( c \)) is equal to the sum of the squares of the lengths of the other two sides (\( a \) and \( b \)).

Mathematically, this is expressed as:

\[ a^2 + b^2 = c^2 \]

Code Snippet for Solving a Right Triangle

Here's a simple Python code snippet that computes the hypotenuse:

import math

def pythagorean_theorem(a, b):
    return math.sqrt(a**2 + b**2)

# Example usage
print(pythagorean_theorem(3, 4))  # Output: 5.0
